# Nova Social Studio Demo

A premium dark social media agency website and local-only admin dashboard built with React and Vite.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown by Vite. The demo dashboard is available at `/admin-demo`.

## Build

```bash
npm run build
```

## Notes

- Demo orders are stored in browser `localStorage`.
- There is no backend, authentication, database, or payment gateway.
- Dashboard orders and metrics are for preview purposes only.
